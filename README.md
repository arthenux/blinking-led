# blinking-led

Blinking LED Arduino Project
A simple "Hello World" project for Arduino that toggles the built-in LED on and off at one-second intervals.

How It Works

The sketch uses Pin 13, which is connected to the internal LED on most Arduino boards (like the Uno, Nano, and Mega).
1. Setup: Sets digital pin 13 as an OUTPUT.
2. Loop:
    * Turns the LED on (HIGH).
    * Waits for 1000 milliseconds (1 second).
    * Turns the LED off (LOW).
    * Waits for another second.

Hardware Needed

* Arduino Board (e.g., Arduino Uno)
* USB Cable to connect to your computer

Getting Started

1. Open the Arduino IDE.
2. Copy the code from BlinkingLed.ino into a new sketch.
3. Connect your Arduino board via USB.
4. Select your board type under Tools > Board.
5. Select the correct port under Tools > Port.
6. Click Upload.
